# 小さな線形代数 CLI を作る（Bun + TypeScript）

※このファイルは `request.md` の日本語版です。

## 目的
agent PR loop が、ワークフローのノートだけでなく「実際に動作し、テストできる」ソースコードを生成できることを示す。

## 作るもの
Bun + TypeScript のプロジェクトを作成し、次を提供すること:

1) `src/` 配下の小さなライブラリ:
- 小さな整数の行列/ベクトルに対して厳密計算ができる程度の有理数（分数）演算。
- 行列/ベクトルのユーティリティ。
- アルゴリズム:
  - 有理数を使ったガウス消去法により、正方線形系 `A x = b` を解く。
  - 2x2 行列の固有値/固有ベクトルと、対角化できるかの判定。
  - 与えられたベクトル集合の張る部分空間への直交射影（可能な限り有理数を使用）。

2) CLI エントリポイント `src/cli.ts`（コマンド）:
- `solve --A <json> --b <json>`
- `eigen2x2 --A <json>`
- `project --basis <json> --b <json>`

入力フォーマット:
- `--A` は配列の配列の JSON（例: `[[1,2],[3,4]]`）。
- `--b` は配列の JSON（例: `[5,6]`）。
- `--basis` はベクトル配列の JSON（例: `[[1,0,0],[0,1,0]]`）。

出力:
- 結果は stdout に JSON として出力する。
- 有理数は、既約分数の文字列（例: `"11/2"` や `"-1"`）で出力する。

3) テスト:
- `test/` 配下にユニットテストを追加（`bun test`）。3つのアルゴリズムすべてと、いくつかのエッジケースをカバーする。
- `bun test` がパスすること。

4) ドキュメント:
- `README.md` に "How to run" セクションを追加し、CLI の実行例と期待する JSON の形を記載する。

## 受け入れ条件
- [ ] `package.json` が存在し、`bun test` がパスする。
- [ ] `src/` にライブラリコードがあり、`src/cli.ts` が CLI を実装している。
- [ ] 少なくとも1つの例で `bun run src/cli.ts solve ...` が動作する。
- [ ] 出力の有理数は既約分数の文字列になっている（厳密に扱えるケースで浮動小数点を使わない）。
- [ ] README が使い方をドキュメント化している。
